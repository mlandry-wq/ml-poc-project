Problématique & Objectif

Contexte médical

Le NICU (Neonatal Intensive Care Unit) accueille les nouveau-nés en détresse : prématurité, complications obstétricales, pathologies maternelles.

  • Coût moyen : 3 000 à 100 000 USD par séjour
  • Ressources hospitalières limitées et spécialisées
  • Charge émotionnelle considérable pour les familles

Objectif de NORA

Identifier dès la grossesse les mères dont le bébé présente un risque élevé d'admission NICU, pour :

  • Intensifier le suivi prénatal ciblé
  • Planifier les ressources néonatales en amont
  • Orienter vers des spécialistes en médecine fœtale
  • Prioriser les interventions de prévention

Problème ML

Classification binaire supervisée

  • 0 — Pas d'admission NICU (91.1%)
  • 1 — Admission NICU · cible à détecter (8.9%)

Le déséquilibre fort rend l'accuracy trompeuse — un modèle naïf atteint 91% en prédisant toujours "non".

Asymétrie des coûts d'erreur

ErreurConséquence
Faux NégatifBébé à risque non détecté → grave, irréversible
Faux PositifHospitalisation inutile → coût financier, stress

→ F-beta β=2 : pénalise les FN deux fois plus que les FP.

Dataset & Collecte

Source

CDC Natality 2018

Base nationale américaine des certificats de naissance, centralisée par le NCHS (National Center for Health Statistics).

  • Dataset original : 3,8 millions de naissances
  • Échantillon utilisé : 99 900 lignes
  • Variables disponibles : 200+ colonnes brutes CDC
  • Variables retenues : 19

Collecte & Encodage

Données issues des certificats de naissance standardisés.

  • Variables déclaratives (tabac, antécédents)
  • Codes CDC techniques → labels métier
  • Sentinelles numériques : 9, 99, 99.9 → NaN
  • Cible filtrée : uniquement Y et N

Défis identifiés

  • Déséquilibre sévère (9% / 91%)
  • Valeurs sentinelles masquant les NaN réels
  • Corrélations faibles des variables de base (< 0.05)
  • Outliers médicaux légitimes (IMC > 50, âge > 45)
  • Variables auto-déclarées → biais de sous-déclaration

Variables sélectionnées (19)

Profil maternel de base (11)

Âge, IMC, tabac avant/T1/T3, mois début suivi, aide WIC, hypertension chronique, diabète chronique, assurance, prise de poids

Variables médicales à fort signal (8)

Pluralité (dplural), éclampsie, PMA, traitement infertilité, ATCD prématuré, HTA gestationnelle, ATCD mort fœtale, diabète gestationnel

Choix des modèles

MÉTRIQUE DE SÉLECTION
Fβ=2  =  5 · Précision · Recall  /  (4 · Précision + Recall)
β=2 → un Faux Négatif est 2× plus pénalisant qu'un Faux Positif

1 · Régression Logistique — Baseline

  • Frontière linéaire dans PCA(10)
  • Pas de SMOTE — threshold seul
  • Entraînement instantané
  • Rôle : plancher de performance

Recall=0.354 · Seuil=0.10

2 · Random Forest ✓ Retenu

  • 500 arbres · balanced_subsample
  • SMOTE + PCA(10)
  • Robuste aux outliers médicaux
  • Capture interactions non-linéaires

Recall=0.766 · F-beta=0.350 · Seuil=0.16

3 · HistGBM + Optuna

  • Boosting séquentiel correctif
  • 25 trials Optuna (TPE) · CV=3
  • Pipeline SMOTE→PCA→HGB dans chaque fold
  • Seuil conservateur (moins de FP)

Recall=0.698 · F-beta=0.337 · Seuil=0.46

Interprétation médicale — Random Forest

  • Sur 100 bébés à risque : 77 détectés, 23 manqués
  • ~8 faux positifs par vrai positif (seuil 0.16)
  • +41 pts de recall vs baseline linéaire
  • AUC-ROC > 0.80 — bonne discrimination globale

HistGBM — profil pratique

  • Recall légèrement inférieur : 69.8%
  • Seuil conservateur (0.46) → moins de FP
  • Meilleur choix en milieu hospitalier à ressources limitées
  • F-beta β=2 proche : 0.337 vs 0.350 pour RF
Note sur la précision faible (0.10) : conséquence directe du seuil bas choisi pour maximiser le recall. Dans ce contexte médical, c'est un compromis intentionnel et documenté — manquer un cas à risque est considéré deux fois plus grave qu'une fausse alerte.

Limites & Pistes d'amélioration

Limites du dataset

  • Pas de données biologiques — bilans sanguins, marqueurs sériques, génétiques : facteurs les plus prédictifs cliniquement → plafond de performance (~76% recall)
  • Biais géographique — population américaine CDC 2018 uniquement
  • Variables auto-déclarées — sous-déclaration possible du tabac et des antécédents

Limites du modèle

  • Interprétabilité réduite — PCA(10) : pas de contribution variable par variable (SHAP nécessaire en production)
  • Faux positifs élevés — ~8 FP par VP avec seuil 0.16
  • SMOTE synthétique — performances réelles en production pourraient être légèrement inférieures
  • Fenêtre temporelle — applicable 2e–3e trimestre uniquement
